# Sales & Demand Forecasting for Businesses

## Overview

This project is developed as part of the Machine Learning Internship at Future Interns.

The objective of this project is to predict future business sales using historical store sales data and machine learning techniques. The project includes data preprocessing, exploratory data analysis, feature engineering, machine learning model training, and deployment using Streamlit.

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/harshitadeheri/FUTURE_ML_01.git
```

---

## Navigate to Project Folder

```bash
cd FUTURE_ML_01
```

---

## Create Virtual Environment

```bash
python -m venv .venv
```

---

## Activate Environment

### Mac/Linux

```bash
source .venv/bin/activate
```

### Windows

```bash
.venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Streamlit App

```bash
streamlit run app.py
```
